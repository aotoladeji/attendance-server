Android App Integration

Point your Android app to the endpoints below using the server's IP address. All endpoints return JSON. Replace YOUR-IP:3000 with your actual server address.

GET
/api/app/session
Get active session info. Call on app launch to confirm a session is active.
GET
/api/app/students
Fetch all students + their attendance status in the active session.
GET
/api/app/check/{studentId}
Check if a specific student has already marked attendance. Use before marking to show status.
POST
/api/app/mark
Mark a student present. Returns 409 Conflict if already marked (duplicate-safe across all devices).
Body: { "studentId": "STU001", "device": "device-name" }
