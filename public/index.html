Android App Integration

Point your Android app to these endpoints using your server's IP address. All endpoints return JSON. Replace YOUR-IP:3000 with your actual server address shown at the top.

GET
/api/app/session
Get active session info. Call on app launch to confirm a session is running.
GET
/api/app/students
Fetch all students + their current attendance status in the active session.
GET
/api/app/check/{studentId}
Check if a specific student has already marked attendance.
POST
/api/app/mark
Mark a student present. Returns 409 Conflict if already marked — duplicate-safe across all devices simultaneously.
Body: { "studentId": "STU001", "device": "tablet-1" }
